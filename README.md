# 🏆 Tournament Tracker

A real-time tournament tracking app built using Flutter and Firebase.

## 🚀 Features

* Live match scores with real-time updates
* Admin panel to update scores instantly
* Team standings leaderboard
* Firebase Authentication for admin access
* Cloud Firestore for real-time database

## 🛠 Tech Stack

* Flutter (Frontend)
* Firebase Authentication
* Cloud Firestore
* Vercel (Deployment)

## 📸 Demo

Live scores update instantly across screens when modified from the admin panel.

## ⚙️ Setup

```bash
flutter pub get
flutter run -d chrome
```

## 🌐 Deployment

```bash
flutter build web
vercel --prod
```

## 📌 Note

Ensure Firebase is properly configured using `flutterfire configure`.
